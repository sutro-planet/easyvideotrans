# API说明
可以重点参考app.py文件中的代码。

## 视频上传
- 说明：上传视频文件，并返回视频ID。
- 路径：/video_upload
- 请求方法：POST
- 返回：
    - 成功：code:200 {"message": \[提示信息\], "video_id": \[视频ID\]}
    - 失败：code:500 { "message": \[错误信息\]}

## youtube视频获取
- 说明：根据视频ID获取下载youtube视频
- 路径：/yt_download
- 请求方法：POST
- 返回:
    - 成功：code:200 {"message": \[提示信息\], "video_id": \[视频ID\]}
    - 失败：code:500 { "message": \[错误信息\]}

## 视频下载
- 说明：根据视频ID下载视频文件
- 路径：/yt/<video_id>
- 请求方法：GET
- 返回：
    - 成功：视频文件
    - 失败：code:404 { "message": "\[错误信息\]}
